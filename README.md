# ROI Calculator

A clean and intuitive ROI (Return on Investment) calculator built with HTML, CSS, and JavaScript.

This project was designed to simulate investment growth over time using a structured, dashboard-style interface. It combines financial calculation logic with a focus on clarity, usability, and visual presentation.

## Live Demo

https://kashpl.github.io/roi-calculator/

## Preview

![ROI Calculator Preview](./preview.png)

## Features

- Investment simulation with monthly compounding
- Initial investment and recurring monthly contribution inputs
- Final balance estimation over time
- Total invested amount calculation
- Net profit analysis
- ROI percentage calculation
- Average monthly return estimation
- Monthly growth projection table
- Clean and responsive dashboard interface

## Why I Built This

The goal of this project was to create a straightforward financial tool that feels practical and easy to use.

Instead of focusing only on isolated calculations, the idea was to present results in a way that makes them easier to understand and more useful in a real-world context.

It was also an opportunity to improve UI structure, readability, and overall user experience.

## Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript  

## How It Works

The calculator uses compound growth logic to simulate how an investment evolves over time.

It considers:
- initial capital  
- monthly contributions  
- interest rate  
- investment duration  

Based on these inputs, the application calculates:
- total invested amount  
- final balance  
- total profit  
- ROI percentage  
- average monthly return
